# Migration Tools

This directory contains all the scripts and documentation used to migrate and reorganize the experiment structure. These files are kept for reference but are no longer needed for day-to-day research.

## Migration Scripts (Historical)
- `move_scripts.py` - Moved experiment scripts from old experiments/ directory
- `move_data.py` - Moved experimental data to organized structure  
- `reorganize_experiments.py` - Renumbered experiments and cleaned up structure
- `final_cleanup.py` - Removed old analysis/ directory
- `cleanup_old_experiments.py` - Removed old experiments/ directory
- `complete_migration.sh` - Bash script for complete migration

## Documentation (Reference)
- `exp_007_discussion.md` - Discussion document for EWMA gamma sensitivity experiment
- `migration_status.md` - Historical migration status tracking
